# PruebaTecnicaAngular

Proyecto generado con [Angular CLI](https://github.com/angular/angular-cli) versión 19.1.6.

## Prerrequisitos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) (recomendado v18 o superior)
- [npm](https://www.npmjs.com/) (incluido con Node.js)
- [Angular CLI](https://angular.dev/tools/cli) (recomendado v19 o superior)

Para instalar Angular CLI globalmente (opcional):
```bash
npm install -g @angular/cli
```

## Cómo obtener el proyecto

### Opción A — Clonar desde Git

```bash
git clone https://github.com/jdep98/prueba-tecnica-angular
cd prueba-tecnica-angular
```

### Opción B — Descargar ZIP

1. Descarga el archivo `.zip` del repositorio desde la plataforma donde esté alojado.
2. Extrae el contenido en una carpeta de tu elección.
3. Abre una terminal y sitúate en la carpeta del proyecto:
```bash
cd ruta/a/la/carpeta/prueba-tecnica-angular
```

## Pasos para levantar el proyecto (comunes a ambas opciones)

1. Instalar dependencias:
```bash
npm install
```

2. Verificar la instalación de Angular (opcional):
```bash
ng version
```

3. Iniciar el servidor de desarrollo:
```bash
ng serve
```

Alternativamente puedes usar:
```bash
npm start
```

4. Abrir la aplicación en el navegador:

- Navega a `http://localhost:4200/`
- La aplicación se recargará automáticamente al modificar los archivos fuente.
